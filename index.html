PROJETO EM DESTAQUE 🚀

Leitor Financeiro Pro

Um gerenciador financeiro reativo e automatizado para Android que lê notificações bancárias em segundo plano.

Recursos Implementados:

  • ✨ Leitura em Background: Captura automática de notificações financeiras com extração de destinatários/remetentes e prevenção robusta contra transações duplicadas consecutivas.
  • 🔄 Controle de Reservas: Módulo de poupança de dinheiro integrado que deduz reservas automaticamente do saldo líquido disponível, com aba de visualização dedicada.
  • 🎨 UI/UX Bottom-Focused: Barra de navegação inferior otimizada para uso com o polegar, botão FAB central expandível via ModalBottomSheet e suporte dinâmico a múltiplos temas (Dark e Clean Claro Azul).
  • 📄 Geração de Comprovantes em PDF: Exportação de relatórios periódicos em formato A4 e compartilhamento instantâneo de comprovantes individuais formatados com código de autenticação único.
  • 🔍 Filtros e Busca Instantâneos: Campo de busca dinâmica acoplado a chips de filtragem temporal (Hoje, Semana, Mês) e dropdown de mapeamento limpo de bancos.
  • 📲 Atualizações Automáticas: Sistema nativo de verificação de novas versões conectado a servidor Flask, capaz de baixar e disparar a instalação do novo APK de forma transparente.

Stack Tecnológica:

Kotlin Jetpack Compose Room Database Coroutines & Flow NotificationListener PDF Document API HttpURLConnection / JSON Flask (Auto-Update Server)

GESTÃO MÉDICA 🩺

ClinicFlow

Um painel integrado de gestão de atendimentos, cirurgias e conciliação inteligente de agenda médica para Android.

Recursos Implementados:

  • 📅 Agenda Mensal Unificada: Calendário interativo integrado diretamente no painel principal para visualização rápida e reativa dos compromissos diários.
  • 🔄 Sincronização com Google Agenda: Integração com a agenda compartilhada do médico via CalendarContract, filtrando apenas compromissos profissionais e ignorando feriados e aniversários automaticamente.
  • ✂️ Limpeza Automática de Histórico: Algoritmo inteligente que mantém salvas no banco de dados local apenas as próximas cirurgias agendadas (a partir da data de hoje).
  • 📸 Relatórios e Anexos de Fotos: Módulo completo para redigir relatórios clínicos pós-atendimento e anexar imagens do processo (ex: marcações pré-cirúrgicas) com galeria dinâmica e zoom.
  • 📊 Métricas de Produtividade: Acompanhamento em tempo real do total de consultas, cirurgias agendadas e quantidade de pacientes únicos no período.

Stack Tecnológica:

Kotlin Jetpack Compose Coroutines & Flow CalendarContract API Material Design 3 MVVM
