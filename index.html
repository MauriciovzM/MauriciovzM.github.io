GPS Pro (Sistema de Navegação Ativa)

Um sistema de mapas e navegação ativo com visual cyberpunk, câmera 3D de seguimento com offset e avisos de manobras por comando de voz.

Recursos Desenvolvidos:

  • 🗺️ Modo Navegação 3D (estilo Waze): Câmera travada no veículo com rotação automática baseada na proa (bearing) e offset trigonométrico para mostrar mais pista à frente.
  • 🔊 Avisos de Manobras passo a passo: Integração com TextToSpeech nativo do Android para comandos de voz em português e recálculo de rota caso o veículo desvie 60 metros.
  • 🔍 Busca Integrada Nominatim: Busca preditiva em tempo real que exibe os detalhes do endereço no rodapé (rua, bairro e cidade) com botão "Ir até aqui".
  • 🚨 Alertas Compartilhados via Firestore: Registro em tempo real de radar, buraco e polícia com sumiço automático do pin após 2 horas no banco de dados.

Stack Tecnológica:

Kotlin osmdroid OSRM API Play Services Location Firebase Firestore TTS API
